Our-ui 是一个基于 React 和 Next.js 的前端组件库，旨在简化和加速 Web 应用程序的开发。这个库是从 shadcn-ui 派生而来的，提供了一系列易于使用和高度可定制的 UI 组件。

使用 npm 或 yarn 安装 Our-ui：

```bash

# or
yarn add our-ui
